# Вопросы по архитектуре проекта

## Общие вопросы
1. Какие маркетплейсы планируются к интеграции помимо Wix?
   - **Ответ**: Allegro, Wix, Amazon(в будущем)
   - **Текущее состояние**: Реализована базовая интеграция с Allegro и Wix
   - **Рекомендации**: Стандартизировать API для упрощения добавления новых площадок

2. Какой ожидаемый объем данных (количество товаров, частота обновлений)?
   - **Ответ**: Частота обновления данных и синхронизация остатков максимальная. список товаров постоянно увеличивающийся.
   - **Текущее состояние**: Асинхронная обработка через Celery
   - **Рекомендации**: 
     - Внедрить кэширование Redis
     - Оптимизировать запросы к БД
     - Добавить партиционирование для больших таблиц

3. Требования к SLA для синхронизации данных?
   - **Ответ**: Пока никаких, внутренний личный проект
   - **Текущее состояние**: Базовая синхронизация
   - **Рекомендации**: Добавить мониторинг времени синхронизации

4. Необходимость поддержки мультитенантности?
   - **Ответ**: Не требуется на данном этапе
   - **Текущее состояние**: Однопользовательская система
   - **Рекомендации**: Архитектура позволяет добавить в будущем при необходимости
